## CI-caught bugs fixed today (all honest catches, documented in commits)

1. `9899c3c`: migrations were resolved from the source-tree path at
   runtime → release container (binary-only image) panicked at startup,
   compose-smoke timed out. Fix: `sqlx::migrate!("./migrations")` (embedded
   at compile time) + the `macros` feature.
2. `703f70d` / `2018304`: db-tests share one job database; kt_leaves.seq is
   one global BIGSERIAL, so parallel tests tripped the leaf-index = seq-1
   drift guard. Fix: per-test PostgreSQL SCHEMA isolation (search_path via
   after_connect) for both KT-touching test files. The §4 guard is
   unchanged and still exact.
3. `92902a8`: hand-written base64 test bodies missed padding → 422
   (correct server behavior). Encode via the base64 crate in tests.
